# anime project

Веб-приложение для сайта аниме.

## Этапы проекта:
1. Инфологическое проектирование
2. Даталогическое проектирование
3. Реализация базы данных PostgreSQL
4. Разработка серверной части (ASP.NET Core)
5. Разработка клиентской части

## Структура:
- docs/ — документация и диаграммы
- database/ — SQL-скрипты
- backend/ — серверная часть
- frontend/ — клиентская часть
